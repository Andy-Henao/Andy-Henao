## 🚀 Featured Projects

### 🏠 [House Price Prediction](https://github.com/kmgomezm/proyecto_estadistica_multivariada)
> **Regression · Multivariate Statistics · Ensemble Methods · SHAP · MLflow**

End-to-end supervised learning pipeline to predict house sale prices (`SalePrice`) from 80 features across 1,460 records. Covers full EDA, preprocessing (missing values, encoding, log-scaling), 5-fold cross-validation, and hyperparameter tuning across 9 models — from regularized linear models (Ridge, Lasso) to boosting ensembles (XGBoost, LightGBM, CatBoost) and an MLP. Includes **SHAP analysis** for feature interpretability and **MLflow** experiment tracking. Best models achieve **R² ≈ 0.90–0.94** and **RMSE ≈ $18K–$25K USD**.

`Python` `scikit-learn` `XGBoost` `LightGBM` `CatBoost` `TensorFlow` `SHAP` `MLflow` `pandas` `Jupyter`

### 💧 [Water Potability Prediction](https://github.com/WonderfulAme/Potability-of-water)
> **Logistic Regression · Statistical Analysis · PCA · Python & R**

Applied logistic regression on 3,276 water samples to predict potability based on physicochemical parameters (pH, turbidity, chloramines, etc.). Performed normality testing (Kolmogorov-Smirnov), Hotelling's T² validation of train/test splits, and PCA-based dimensionality reduction. Model reached **AUC ≈ 0.75** with interpretable coefficient analysis.

`Python` `R` `scikit-learn` `PCA` `Statistical Testing` `Logistic Regression`
